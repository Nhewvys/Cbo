Quem Sou Eu

Olá! Meu nome é Nhewvys, e minha jornada na programação começou em 2020, durante o ensino médio técnico. Na época, ainda estava descobrindo meu interesse pela área, mas foi em 2022, durante as aulas de desenvolvimento web, que tudo mudou. Foi ali que me apaixonei por criar soluções tecnológicas.

Um dos momentos mais marcantes foi o desenvolvimento de um projeto que visava organizar as filas do refeitório usando automação com Arduino. Esse projeto não só foi selecionado para a FECIBA (Feira de Ciências da Bahia), em Salvador, como também conquistamos o primeiro lugar! Desde então, tenho me aprofundado cada vez mais nos estudos, buscando sempre aprender e evoluir, mesmo com os desafios impostos pela pandemia.

Hoje, atuo como desenvolvedor web e designer gráfico, criando sistemas personalizados que resolvem problemas reais e impulsionam negócios. Minha paixão é transformar ideias em soluções funcionais e eficientes.

O Que Faço

Ofereço serviços de desenvolvimento web, design gráfico e criação de sistemas de gerenciamento e CRM. Trabalho com tecnologias modernas como HTML, CSS, JavaScript, Node.js, React e bancos de dados SQL. Meus projetos são focados em usabilidade, desempenho e design responsivo.

Projetos Desenvolvidos

Bot de WhatsApp em Node.js

Desenvolvimento de um bot para WhatsApp com funcionalidades de automação, integração de APIs e envio de mensagens automatizadas.

Automação para Controle de Filas

Projeto de automação com Arduino para controle e organização de filas em ambientes como refeitórios ou eventos.

Sistema de Gerenciamento de Contatos

Desenvolvimento de um sistema simples para gerenciar contatos, com funcionalidades de adição, exclusão e atualização de dados.

Dashboard de Análise de Dados

Criação de um painel de controle para análise e visualização de dados em tempo real, utilizando gráficos e integração com bancos de dados.

Sistema de Inventário

Desenvolvimento de um sistema de gerenciamento de inventário para controle de estoque, com funcionalidades de entradas e saídas de produtos.

Portfólio Interativo

Criação de um portfólio online interativo, utilizando HTML, CSS e JavaScript, com animações e transições dinâmicas para uma experiência do usuário mais envolvente.
